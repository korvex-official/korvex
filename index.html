// L UNIVERS KORVEX
// QUI EST CALVASSE ?

A la base je suis un joueur DayZ avec une imagination debordante. J aime explorer tous les outils qu internet nous offre — c est dans cet esprit que j ai cree KORVEX, ma maison mere, et ses branches pour travailler sur tout ce qui me passionne.

C est dans cette optique que le premier tome de ma trilogie inspiree de l univers DayZ a vu le jour : Protocole P9. Je n avais jamais envisage d ecrire un jour et je n ai aucune formation litteraire, mais je deborde d idees — et quoi de mieux que de les poser dans une histoire. Un semi-lore DayZ sur fond d une fiction entierement inventee.

D autres creations verront le jour, mais pour les decouvrir il faudra me suivre dans l aventure.

// CALVASSE — FONDATEUR DE KORVEX
// PHILOSOPHIE KORVEX

KORVEX a ete cree pour la communaute DayZ mais aussi pour les non-joueurs, avec une philosophie de gratuite totale pour tous. Pas de pub intempestive, pas de paywall, pas de PDF basique, pas de plateforme envahie de publicites.

KORVEX est une vision a long terme qui compte s installer partout sur le net — une vision ambitieuse qui me pousse a travailler chaque projet au maximum. Que vous soyez joueur DayZ, admin de serveur ou simple internaute, KORVEX fera son maximum pour vous satisfaire. C est la communaute qui nous fera atteindre les sommets.
